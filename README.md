#   تذكر اسم المشروع
نظام إدارة صيدلية 
##  وصف المشروع

نظام يهدف إلى تسهيل إدارة الصيدليات من خلال تنظيم الأدوية، متابعة المخزون، وتسجيل عمليات البيع بشكل دقيق وسريع.

---

##  مميزات النظام
مثلا:

-  إدارة الأدوية (إضافة / تعديل / حذف)
-  تسجيل عمليات البيع
-  متابعة المخزون
-  تنبيه للأدوية القريبة من انتهاء الصلاحية
-  البحث عن الأدوية بسهولة

---

##  التقنيات المستخدمة
مثلا:
- لغة البرمجة: (python)
- قاعدة البيانات: ( SQLite)
- أدوات أخرى: (إن وجدت)

---

##  طريقة تشغيل المشروع
تشرح خطوات تشغيل النظام :
1. قم بتحميل المشروع
2. افتح الملفات باستخدام (فيجوال ستديو )
3. شغّل الملف الرئيسي
4. ابدأ باستخدام النظام
